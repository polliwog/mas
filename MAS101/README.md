#### 说明
1. NUCLEO-F072RB

![image](NUCLEO-F072RB.png)

1.1 USB
		DP: PA12
		DM: PA11

1.2 RS485 (USART4)
		TX: PC10
		RX: PC11
		DE: PA15
		
1.3 CAN
		TX: PB9
		RX: PB8
